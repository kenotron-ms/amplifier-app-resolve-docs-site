**Resolve** is Amplifier's autonomous code-generation platform. You submit a
task as an **instance**, a pluggable **resolver** executes it in an isolated
container, and you watch progress through a live **event stream** — optionally
verifying the result against acceptance criteria with a **reality check**.

Resolve is a multi-repo system: a FastAPI backend, a React frontend, and a set
of resolver plugins (each with its own viewport UI). This section documents how
to build on it and operate it.
